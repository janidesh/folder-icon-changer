# 📁 Folder Icon Changer

A simple yet powerful tool to customize your folder icons easily.

---

## 🚀 About the Project

This project started as something I built purely for **personal use** — I just wanted an easy way to customize folder icons without dealing with complicated tools or manual steps.

But after using it for a while, I realized…
👉 *this is actually pretty useful.*

So I decided to clean it up and **publish it for others to use as well.**
